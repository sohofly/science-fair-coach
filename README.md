# 科展小助手 Science Fair Coach

協助小學生從生活觀察與個人興趣出發，找到安全、可測量、可在期限內完成的科展題目。

目前包含可離線試用的前端，以及可部署的Supabase正式後端。正式後端支援Google教師登入、班級加入碼、學生代號與PIN、思考歷程時間軸、資料匯出、保留政策及選用的動態AI推薦。

## 已確認方向

- 繁體中文、手機和平板優先
- 活潑的兒童網站風格
- 支援物理、化學、生物、地科、數學、應科一、應科二、應科三
- 八個組別由學生選擇，而且可以隨時更換
- 前三步（組別、興趣／觀察、候選題目）均可返回修改
- 保留教師／家長檢視
- 題目選定前進行安全與可行性檢查
- 內建八科共 17 個起始題目
- 自動保存進度，並可列印教師檢視摘要
- 推薦同時考慮所選科別與學生興趣；不吻合時提出替代科別
- 部分題目提供科教館歷屆作品、前人研究差異與創新發展建議
- 教師Google登入、班級與多位學生管理
- 追加式思考歷程、教師留言、JSON／CSV／PDF匯出
- 365天後唯讀、30天匯出期、第395天自動刪除
- 動態AI多輪問答、近兩年議題與科教館作品搜尋（需後端API金鑰）

## 隱私與限制

- 不要求登入或輸入學生姓名
- 未設定Supabase時自動維持本機試用模式
- 動態AI結果仍需教師核對來源、科別、安全與可行性

正式部署請見[`SETUP.md`](SETUP.md)。
