# 科展小助手 Science Fair Coach

協助小學生從生活觀察與個人興趣出發，找到安全、可測量、可在期限內完成的科展題目。

目前為可正式試用的第一版。學生可完成科別選擇、條件填寫、候選題目比較、研究摘要與教師檢視。資料只儲存在使用者目前的瀏覽器，不會上傳學生資料。

## 已確認方向

- 繁體中文、手機和平板優先
- 活潑的兒童網站風格
- 支援物理、化學、生物、地科、數學、應科一、應科二、應科三
- 八個組別由學生選擇，而且可以隨時更換
- 前三步（組別、興趣／觀察、候選題目）均可返回修改
- 保留教師／家長檢視
- 題目選定前進行安全與可行性檢查
- 內建八科共 17 個起始題目
- 自動保存進度，並可列印教師檢視摘要
- 推薦同時考慮所選科別與學生興趣；不吻合時提出替代科別
- 部分題目提供科教館歷屆作品、前人研究差異與創新發展建議

## 隱私與限制

- 不要求登入或輸入學生姓名
- 尚未連接 AI 後端；推薦內容來自網站內建題目庫
- 文獻仍需由學生與教師依網站提供的下一步自行查證
